## 📖 Tentang Proyek

**SEBEL Mobile App** merupakan aplikasi sekolah yang dikembangkan untuk mempermudah proses belajar mengajar serta akses informasi sekolah melalui perangkat mobile.

Fitur utama aplikasi meliputi:

* Informasi siswa
* Materi pembelajaran
* Tugas dan penilaian
* Pengumuman sekolah
* Jadwal kegiatan akademik
* Notifikasi informasi terbaru

## 🚀 Instalasi Project

### 1. Membuat Folder Project

Buat folder baru untuk menyimpan source code aplikasi.

```bash
mkdir sebel-mobile
cd sebel-mobile
```

### 2. Buka Terminal

Masuk ke terminal pada folder yang telah dibuat.

```bash
cd sebel-mobile
```

### 3. Clone Repository

Clone source code dari repository yang telah disediakan.

```bash
git clone <LINK_REPOSITORY_SEBEL_APP>
```

Contoh:

```bash
git clone https://github.com/username/sebel-app.git
```

### 4. Masuk ke Folder Project

```bash
cd sebel-app
```

### 5. Install Dependency

```bash
npm install
```

atau

```bash
yarn install
```

### 6. Jalankan Aplikasi

```bash
npm start
```

atau

```bash
yarn start
```
